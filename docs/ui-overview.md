# UI 设计稿分析

> 文档基于用户 2026-07-20 提供的 29 张截图整理，用于后续需求讨论和开发参考。

## 一、整体布局

- **窗口标题**：网络测试工具箱 V8.6 | Network Testing Toolkit
- **副标题**：Professional Network Diagnostic & Analysis Tool
- **布局结构**：左侧固定导航栏 + 右侧主内容区 + 底部状态栏
- **主题色**：蓝色系（#2196F3 为主色），绿色表示正常/开始，橙色/红色表示警告

## 二、左侧导航菜单

| 一级菜单 | 二级菜单 |
|---------|---------|
| 概览 | 仪表盘 |
| 诊断检测 | 网络健康、Ping 测试、路由追踪、端口扫描、主机发现、摄像头扫描、安全自测、日志审计 |
| 速度测试 | 外网测速、内网测速、会话测试 |

| 实用工具 | 子网计算、IP 信息检测、MAC 地址、WiFi 密码、路由表、连接测试 |
| 网络服务 | 网络服务、本机设置、防火墙配置 |
| 监控运维 | 链路监控、流量监控 |
| 远程工具 | 远程终端、远程桌面、串口调试 |
| 智能系统 | AI 网络助手、系统设置、关于和支持 |

## 三、页面功能拆解

### 3.1 网络状态总览（仪表盘）

顶部展示 6 个核心指标卡片：

- CPU 使用率（带实时折线图）
- 内存占用（带进度条）
- 磁盘占用（带进度条）
- 网络吞吐（上行/下行速率）
- 系统运行时长
- 进程数量

中部展示连通性状态：

- 网关延迟
- DNS 延迟
- 互联网延迟

底部左右分栏：

- 左侧：系统网络事件日志（时间 + 事件 + 结果）
- 右侧：工具使用说明（按功能列出说明）

最底部：快速启动栏（Ping、路由追踪、端口扫描、IP 信息、网速测试）

### 3.2 网络健康检查

- 标签页：基本健康检查、单独环路检测
- 操作按钮：开始健康检查、清空结果
- 检测项目清单：
  1. 网卡状态检测
  2. 网关连通性测试
  3. DNS 解析速度测试
  4. 外网连接测试
  5. 网络延迟评估
  6. 网络速度测试
  7. 网络环路检测（基础）
- 输出区域：综合评分 + 优化建议

### 3.3 网络环路检测

- 网卡选择下拉框 + 刷新按钮
- 操作按钮：开始环路检测、清空结果
- 检测方法说明：以太网帧检测（底层检测）
- 知识说明：什么是网络环路、常见原因（端口错误连接、网线误接形成环路、未启用 STP 协议）

### 3.4 Ping 测试

- 标签页：单个 Ping、持续 Ping、批量 Ping、网段 Ping、TCP Ping
- 配置项：
  - 目标主机（输入框 + 历史下拉）
  - 测试次数
  - 数据包大小（字节），快捷按钮 32B / 1KB / 4KB / 8KB
- 操作按钮：开始 Ping、停止
- 输出区域：Ping 测试结果 + 底部统计（已发送 / 接收 / 丢失 / 丢包率）

### 3.5 路由追踪

- 配置项：
  - 目标主机
  - 最大跳数
- 操作按钮：开始追踪、停止追踪、网络诊断
- 输出区域：路由追踪结果列表

### 3.6 摄像头扫描

- 扫描参数：
  - 网络范围（如 `192.168.1.0/24`）
  - 扫描品牌：海康威视、大华、宇视、天地伟业、通用
  - 扫描模式：快速扫描、深度扫描、ONVIF 发现
  - 线程数、超时（秒）
- 操作按钮：开始扫描、停止扫描、清空结果、导出清单
- 统计：扫描 IP 数、已检测数、发现数、进度
- 输出区域：扫描结果列表
- 右侧扫描说明：
  - 支持协议（HTTP/HTTPS、ONVIF、RTSP、常见品牌通用端口）
  - 扫描限制（仅标准协议、私有协议无法识别、需认证设备可能无法检测）
  - 扫描建议（优先用通用模式、ONVIF 兼容性最好、建议在局域网内扫描）

### 3.7 主机发现

- 标签页：扫描结果、局域网拓扑、远程唤醒
- 扫描参数：
  - 网络范围（如 `192.168.1.0/24`）
  - 扫描方式：Ping 扫描（快速）、TCP 连接（推荐）、混合模式（全面）
  - 线程数、超时（秒）
- 操作按钮：开始扫描、停止扫描、导出结果
- 统计：总 IP、已扫、存活、进度
- 输出区域：扫描结果列表

### 3.8 本机安全自测

- 顶部 4 个统计卡片：综合评分、高危项、中危项、安全项
- 操作按钮：导出报告、开始检测
- 检测范围（共 12 项）：
  - 防火墙状态
  - SMBv1 危险协议
  - UAC 用户账户控制
  - Guest 账户
  - 远程桌面（RDP）
  - 密码最小长度
  - 账户锁定策略
  - Windows 自动更新
  - 屏幕自动锁定
  - 本地共享目录
  - 高危端口开放
  - 系统版本信息
- 输出区域：检测详情 + 结果分类

### 3.9 本机安全日志审计

- 顶部筛选：时间范围（最近 24 小时等）、最大条数（2000 等）
- 顶部 4 个统计卡片：事件总数、登录成功、登录失败、高危事件
- 操作按钮：导出报告、开始审计
- 分析内容：
  - 登录成功 / 失败统计、可疑暴力破解来源
  - 账户管理事件（新建/删除/改密/锁定）
  - 审计日志清除、审计策略变更等高危操作
  - 各类安全事件按类型汇总
- 提示：读取安全日志通常需要以管理员身份运行本软件
- 输出区域：审计分析详情

### 3.10 内网测速

- 页面说明：测试局域网内两台电脑之间的网络速度，左侧为服务端模式，右侧为客户端模式，支持上传/下载/双向测试
- 服务端模式：
  - 监听端口（默认 5201）
  - 本机网卡选择 + 刷新按钮
  - 提示：服务端监听所有网卡（0.0.0.0），上方显示本机可用 IP 供客户端连接
  - 操作按钮：启动服务端、停止服务端
  - 输出区域：服务端状态
- 客户端模式：
  - 服务器 IP、服务器端口（默认 5201）
  - 测试时长（秒）
  - 测试模式：上传、下载、双向
  - 测试引擎：内置引擎、iPerf3（检测中...）
  - 操作按钮：开始测速、停止测试
  - 输出区域：测试结果

### 3.11 会话数测试

- 测试配置：
  - 测试服务器：主服务器、备用服务器
  - 延迟（ms）
  - 线程数
  - 最大连接数
  - 失败上限
- 操作按钮：开始测试、停止
- 实时统计：
  - 成功连接
  - 失败次数
  - 运行时间
- 输出区域：运行日志

### 3.12 数据包抓包

- 抓包参数配置：
  - 选择网卡（如 以太网 192.168.0.150）
  - 过滤器（示例：tcp | udp | icmp | host 192.168.1.1 | port 80 | tcp port 443）
  - 最大包数
- AI 自定义提示词（可选）：
  - 示例 1：重点分析是否存在端口扫描、异常外联和可疑 DNS 请求
  - 示例 2：请按现象-原因-建议三段式输出，给出高/中/低优先级
  - 示例 3：只关注 192.168.1.0/24 到公网的 TCP 443 流量特征
- 操作按钮：开始抓包、停止抓包、清空结果、导出 PCAP、导入 PCAP、AI 智能分析
- 输出区域：抓包结果 + 抓包统计

### 3.13 协议分析（网络流量分析）

- 顶部配置：
  - 网卡选择（全部）
  - 刷新、开始抓包、停止、清空按钮
  - 状态：就绪 | 需要 scapy + npcap
  - 说明：实时捕获本机进出流量，展示通信内容、协议类型和占比。不同协议颜色标识（蓝=TCP 传输、绿=UDP 传输、黄=DNS 域名、粉=HTTP 网页、紫=Ping、青=ARP）
- 实时通信记录表格：
  - 列：#、时间、来源、去往、类型、大小(B)、内容摘要
  - 每行 = 一次数据包收发，最近 600 条
- 下方左右分栏：
  - 左侧：流量类型占比（各类协议抓到多少包）
  - 右侧：占用流量最多的连接（谁在和谁通信、用的什么协议），列：来源、目标、类型、包数、流量

### 3.14 子网计算

- 标签页：IPv4、IPv6、VLSM、子网划分、路由汇总、反掩码
- IPv4 网络参数输入：
  - 网络地址（如 `192.168.1.0/24`）
  - 操作按钮：开始计算
- IPv4 计算结果：
  - 网络地址
  - 广播地址
  - 子网掩码
  - 主机数量
  - 可用主机
  - 网络类别

### 3.15 IP 信息检测

- 标签页：IP 信息查询、IP 冲突检测
- IP 冲突检测：
  - 选择网卡（自动选择）
  - IP 范围（如 `192.168.1.1-254` 或 `192.168.1.0/24`）
  - 操作按钮：开始检测、停止
  - 检测结果表格列：IP 地址、MAC 地址、状态、冲突次数、冲突 MAC 地址

### 3.16 MAC 地址工具

- MAC 地址输入框
- 操作按钮：查询厂商、格式转换、查看 ARP 表、清除
- 转换格式选项：
  - 冒号分隔（xx:xx:xx:xx:xx:xx）
  - 横线分隔（xx-xx-xx-xx-xx-xx）
  - 点分隔（xxxx.xxxx.xxxx）
  - 无分隔（xxxxxxxxxxxx）
- 输出区域：查询结果 + 功能说明

### 3.17 路由表

- 路由操作：
  - 查看路由表按钮
  - 目标网络、掩码、网关输入框
  - 添加路由、删除路由按钮
- 路由表展示：
  - 列：网络目标、网络掩码、网关、接口、跃点数

### 3.18 连接测试

- 标签页：DNS 查询、DNS 优选、站点测试、Telnet 端口、MTU 测试、SSL 检查、密码生成、NSLookup、连接统计、DNS 污染、端口进程
- DNS 查询：
  - 域名输入框
  - 查询按钮
  - 查询结果输出区域

### 3.19 网络服务

- 标签页：TFTP 服务器、FTP 服务器、Syslog 服务器、HTTP 文件分发、DHCP 服务器
- TFTP 服务器配置（示例）：
  - 监听地址（默认 0.0.0.0）
  - 监听端口（默认 69）
  - 根目录选择
  - 操作按钮：启动 TFTP 服务器、停止服务器
  - 右侧：服务器日志
  - 下方：使用说明、注意事项与风险提示

### 3.20 本机设置

- 系统工具快捷入口网格：
  - 打开 CMD 控制台
  - 打开 PowerShell
  - 管理员 CMD
  - IP 地址设置
  - 关闭防火墙 / 开启防火墙 / 防火墙状态
  - 设备管理器
  - 任务管理器
  - 系统信息
  - 注册表编辑器
  - 事件查看器
  - 磁盘管理
  - 服务管理
  - 资源监视器
  - 计算机管理
  - 控制面板
  - 远程桌面
  - 修改 3389 端口 / 恢复 3389 端口
  - 网络修复工具
  - 网络诊断
  - 性能监视器
  - 网络设置
- 下方操作结果区域：显示每个工具的功能说明和安全提示

### 3.21 防火墙配置

- 标签页：快捷配置、高级规则、规则管理
- 快捷配置：
  - 规则名称、方向（in/out）、动作（allow/block）、配置文件（any）
  - 应用路径、添加应用规则
  - 端口/协议、添加端口规则
  - 一键模板：远程桌面、网页服务、FTP、查看状态
  - 模板选择 + 套用 + 一键添加
  - 操作说明：in=入站，out=出站；allow=允许，block=拦截；需管理员权限
- 高级规则：
  - 五元组配置：源 IP、目的 IP、源端口、目的端口
  - 绑定应用
  - 添加五元组规则
  - 域名规则：输入域名，添加域名规则（先解析为 IP 再下发到防火墙）
- 规则管理：
  - 常用管理操作：查询规则、启用规则、禁用规则、删除规则、开启防火墙、查看状态
  - 提示：命令失败请以管理员身份运行
- 右侧：防火墙操作日志

### 3.22 链路监控

- 监控配置：
  - 监控目标列表（每行一个 IP/域名，如网关、DNS、业务 IP）
  - 阈值与通知：
    - 采样间隔（秒）
    - 超时（ms）
    - 延迟阈值（ms）
    - 丢包阈值（%）
    - 抖动阈值（ms）
  - 告警方式：飞书告警、企微告警
  - Webhook 输入框 + 测试按钮
- 操作按钮：启动监控、停止监控、清空日志、导出 HTML、导出 JSON
- 实时链路状态表格：
  - 列：监控目标、状态、平均延迟、丢包率、抖动、最近一次
- 下方：告警与事件日志

### 3.23 串口调试

- 串口连接配置：
  - 串口（如 COM6）
  - 波特率（默认 9600）
  - 数据位（默认 8）
  - 停止位（默认 1）
  - 校验（默认 None）
  - 回车（CR+LF）
  - 快速预设选择
  - 连接按钮
- 会话页签：欢迎、COM6
- 终端区域：
  - 清屏、关闭、断开按钮
  - 连接状态显示（已连接 COM6）
  - 黑色背景终端，显示设备登录回显和命令交互
- 用途：连接交换机、路由器等串口设备进行命令行调试

### 3.24 远程桌面（RDP）

- 连接参数：
  - 主机 / IP

  - 用户名（默认 Administrator）

  - 分辨率选择（全屏等）
- 操作按钮：连接、清空
- 右侧连接历史：
  - 表格列：主机/IP、端口、用户名、最近连接
  - 操作：连接选中、填入表单、删除选中
  - 提示：双击历史记录直接连接

### 3.25 远程终端

- 标题：SSH / Telnet 多会话终端
- 操作按钮：+ SSH 新建、+ Telnet 新建
- 默认欢迎页：
  - 显示"远程工具箱"
  - 提示支持 SSH / Telnet 多会话并行连接
  - 提示点击右上角按钮开始
